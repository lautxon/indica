## 📤 Cómo subir una nueva canción (sin código)

### Paso 1: Ingresar a GitHub
1. Ve a: https://github.com/lautxon/indica
2. Inicia sesión con tu cuenta de GitHub

### Paso 2: Navegar a la carpeta de canciones
1. Haz clic en: `src/` → `content/` → `canciones/`
2. Verás los archivos `.md` de las canciones publicadas

### Paso 3: Crear nueva canción
1. Haz clic en **"Add file"** → **"Create new file"**
2. En el nombre del archivo escribe: `mi-cancion.md`
3. Pega esta plantilla y completa los campos:

```yaml
---
title: "Título de tu canción"
image: /indica/images/uploads/portada.jpg
audio_url: https://spotify.com/track/xxx
date: 2026-03-13
---

# Descripción / Letras

Escribe aquí la descripción, letras o notas de tu canción.
Puedes usar **negrita**, *cursiva*, listas, etc.
